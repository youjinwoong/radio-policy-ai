# 요약

이 규정은 「행정규제기본법」에 따라 방송미디어통신위원회(방미통위) 소관 규제의 정비를 효율적으로 추진하고, 신설·강화 규제에 대한 심의 등 필요한 사항을 정한 훈령이다. 규제사무 처리의 일반 절차법으로서 규제정비계획 수립, 네거티브(원칙허용·예외금지) 규제전환, 규제심사(사전검토→규제영향분석/규제비용분석→입법·행정예고→자체심사→규제합리화위원회 심사) 절차, 규제순비용 관리, 존속기한·재검토기한 설정, 방송미디어통신규제심사위원회의 구성·운영, 규제개혁 역량강화(교육·홍보·포상)를 규율한다. 총 5개 장 24개 조와 부칙, 별표(규제심사 절차)·별지 1(사전검토 요청서)로 구성된다. 제28호는 위원회 개편에 따른 일부개정으로, 발령한 날(2026. 4. 21.)부터 시행한다.

- 시행일: 2026. 4. 21.
- 공포/개정: 방송미디어통신위원회훈령 제28호, 2026. 4. 21. (일부개정)
- 소관부처: 방송미디어통신위원회(행정법무담당관)
- 상위 근거: 「행정규제기본법」, 「법제업무 운영규정」 제25조제2항(행정규칙 행정예고·법제처 검토)
- 실무영향: 방미통위 소관 법령·행정규칙 제·개정 시 적용되는 내부 규제심사 절차로, 규제 신설·강화 시 규제영향분석서 작성·규제순비용 관리·규제심사위원회 심사를 거쳐야 하며 이해관계자(사업자 포함) 의견제출 창구가 된다.

# 적용 범위

- 적용 대상: 방송미디어통신위원회의 규제사무 처리 전반. 다른 규정에서 특별히 정한 경우를 제외하고 이 규정을 따른다.
- 주체: 행정법무담당관(총괄·간사), 소관 법령 및 행정규칙 소관부서의 장, 방송미디어통신규제심사위원회, 규제합리화위원회(국무조정실), 법제처, 한국개발연구원(KDI) 규제연구센터(비용분석 검증) 등.
- 대상 규제유형: 신설·강화 규제(규제영향분석서 작성), 폐지·완화 규제(규제비용분석서 작성), 규제심사 비대상 규제.

# 주요 내용(구조화)

- 제1장 총칙 (제1조~제2조): 목적(행정규제기본법 근거 규제정비·심의), 적용범위(규제사무 처리의 일반규정)
- 제2장 규제정비계획 수립 (제3조~제4조): 매년 규제개혁과제 발굴 → 규제정비계획 수립·규제합리화위원회 제출(제3조); 제·개정 시 '원칙허용·예외금지' 네거티브 방식 원칙, '우선허용·사후규제' 등 활용(제4조)
- 제3장 규제심사 (제5조~제10조):
  - 제5조 규제심사 대상 여부 사전검토(별지 제1호 서식)→국무조정실 검토요청→규제정보화시스템(ris.go.kr) '대상여부 확인'. 신설·강화 시 규제영향분석서, 폐지·완화 시 규제비용분석서 작성.
  - 제6조 규제영향분석서 등 작성·공표(e-규제영향분석시스템 활용, 입법·행정예고 시 함께 게재)
  - 제7조 규제비용 관리: 규제순비용 발생 시 상응하는 규제 정비, 원칙적 동시 제출(부득이 시 1년 이내 계획 제출로 갈음), 표준형RIA는 KDI 검증·필요 시 비용적정성 심의, 반기별 관리현황 공표
  - 제8조 규제심사 요청: 예고기간 만료 후 의견 반영→자체 규제심사→규제합리화위원회 요청. 행정규칙은 사전 법제처 검토안 첨부.
  - 제9조 규제심사: 개선권고 반영 여부 결정·위원장 보고, 재심사 요청 가능, 예비심사로 중요/비중요규제 구분(중요규제 기준: 직간접비용 100억원 이상 또는 피규제자 100만명 이상, 진입·경쟁 제한적 규제, 사회경제적 부작용 우려 규제)
  - 제10조 존속기한(5년 이내)·재검토기한(3년 이내) 설정 원칙 및 연장 절차
- 제4장 방송미디어통신규제심사위원회 운영 (제11조~제18조): 위원회 기능(제11조), 구성(공동위원장 2인 포함 10~15인, 민간위원 과반수 이상 — 제12조), 위원 임기(민간 2년·1회 연임 — 제13조), 회의(재적 과반 출석·출석 과반 찬성, 대면 원칙 — 제14조), 위원장 직무·소집 통지(5일 전 — 제15조), 간사(행정법무담당관 — 제16조·제17조), 수당(제18조)
- 제5장 규제개혁 역량 강화 (제19조~제24조): 규제신속확인 제도(제19조), 규제혁신 평가(제20조), 규제개혁 교육(제21조), 홍보(제22조), 공무원 책임 면제(고의·중과실 없을 시 — 제23조), 포상(제24조)
- 부칙 <제28호, 2026.4.21.>: 제1조 시행일(발령일 시행), 제2조 위원회 개편에 따른 위원 임기 경과조치

# 별표·서식 목록

- [별표] 규제심사 절차 — '신설·강화' 안건과 '폐지·완화' 안건의 규제심사 흐름도(대상여부 확인→규제영향/비용분석서 작성→위원회 보고→입법·행정예고→확인증 발급→비용분석 검증→부처 자체심사→규합위 예비심사(중요/비중요)→본회의 상정·심사종결→규제 등록/규제편익 적립). 행정규칙안은 행정예고(20일)·법제처 검토를 규합위 예비심사 전까지 완료.
- [별지 1] 규제심사 대상여부 사전검토 요청서 — 법령명, 제·개정 주요내용(신설·강화/폐지·완화 해당 여부), 행정조사 포함여부, 신산업 규제 해당 여부, 입법(행정)예고 기간, 시행일, 국정과제 연계, 예상 이해관계자, 타 법률 관계, 소관부처·담당자. 붙임: 조문별 규제심사 대상여부 검토표.

# 실무 체크리스트

- [ ] 매년 규제개혁과제를 발굴하고 규제정비계획을 수립해 규제합리화위원회에 제출.
- [ ] 규제 제·개정 시 '원칙허용·예외금지' 네거티브 방식을 우선 검토.
- [ ] 회의 상정 전 행정법무담당관에게 별지 제1호 서식으로 규제심사 대상 여부 사전검토 의뢰.
- [ ] 규제정보화시스템(ris.go.kr) e-규제영향분석시스템으로 규제영향분석서(신설·강화) 또는 규제비용분석서(폐지·완화) 작성.
- [ ] 입법·행정예고 시 규제영향분석서를 반드시 함께 게재(행정규칙은 20일 행정예고).
- [ ] 규제순비용 발생 시 상응하는 폐지·완화 규제를 원칙적으로 동시 제출(부득이 시 1년 이내 계획).
- [ ] 표준형RIA 대상은 KDI 규제연구센터 비용분석 검증(필요 시 비용적정성 심의) 신청.
- [ ] 행정규칙 규제심사 요청 시 사전에 법제처 검토안 첨부.
- [ ] 예고기간 종료 후 이해관계인 의견 반영, 자체 규제심사(규제심사위원회 개최) 후 규제합리화위원회 심사 요청.
- [ ] 개선권고 수용 여부 결정·위원장 보고, 필요 시 재심사 요청.
- [ ] 신설·강화 규제에 존속기한(5년 이내) 또는 재검토기한(3년 이내) 설정.
- [ ] 규제심사 후 법령안 내용 변경 시 규제실과 대상여부 재협의.
