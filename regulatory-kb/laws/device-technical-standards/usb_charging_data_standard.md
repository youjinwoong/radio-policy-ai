# 요약

「방송통신발전 기본법」제28조제9항에 근거하여, 자원 낭비 방지와 소비자 편익 제고를 위해 **유선으로 충전하거나 유선으로 충전과 데이터 전송을 동시에 하는** 모바일·스마트기기 등 무선 방송통신기자재의 커넥터를 **USB C형 리셉터클 커넥터로 통합**하도록 정한 과학기술정보통신부고시이다(이른바 USB-C 단일 충전 규격).

본문은 4개 조문(목적·적용대상·적용기준·재검토기한)과 부칙으로 매우 간결하며, 실질 규격은 [별표 1] USB C형 리셉터클 커넥터 규격(외피 개구부 치수, 텅 치수, 신호 접점 치수 등)에 도면과 함께 명시되어 있다. 발령(2025-11-05) 후 1년이 경과한 날인 **2026-11-06 시행**이다.

# 적용 범위

- 근거·목적(제1조): 「방송통신발전 기본법」제28조제9항, 자원 낭비 방지 및 소비자 편익 등.
- 적용 대상(제2조): 유선 방식으로 충전하거나, 유선방식으로 충전과 데이터 전송을 동시에 하며 아래 각 호 어느 하나에 해당하는 모바일·스마트기기 등 무선 방송통신기자재("대상기자재").

  2. 태블릿
  3. 디지털 카메라
  4. 헤드폰
  5. 헤드셋
  6. 휴대용 비디오 게임기
  7. 휴대용 스피커
  8. 전자책리더

- 적용 기준(제3조): 대상기자재는 [별표 1]의 **USB C형 리셉터클 커넥터**를 갖추어야 한다.
- 주의: "유선 충전(또는 유선 충전+데이터 동시)"이 요건이므로, 무선 충전 전용 기기 등은 문언상 유선 충전 요건 대상에 해당하는지 개별 확인이 필요.

# 주요 기술기준(구조화)

- **제1조(목적)**: 자원 낭비 방지·소비자 편익을 위한 충전 및 데이터 전송 방식 기술기준.
- **제2조(적용 대상기자재)**: 13개 품목(위 목록).
- **제3조(적용기준)**: [별표 1] USB C형 리셉터클 커넥터 구비 의무.
- **제4조(재검토기한)**: 「행정규제기본법」및 「훈령·예규 등의 발령 및 관리에 관한 규정」에 따라 2025-01-01 기준 매 3년이 되는 시점(매 3년째 12월 31일까지)마다 타당성 검토.
- **부칙(제2025-56호, 2025.11.5.)**
  - 제1조(시행일): 발령 후 1년이 경과한 날부터 시행 → 2026-11-06.
  - 제2조(다른 고시의 개정): 「방송통신기자재등의 적합성평가에 관한 고시(국립전파연구원고시)」[별표 1] 적합성평가 대상기자재(제3조 관련) 제7호다목 25) "접속 커넥터"에 **USB C형 리셉터클 커넥터** 항목을 추가(적합성평가유형: 자기적합확인, 기기부호 USBCR, 적용분야 유선).

## [별표 1] USB C형 리셉터클 커넥터 규격 (제3조 관련)

도면 4개(외피 개구부 너비/높이, 리셉터클 접촉 치수 및 대체 치수)와 주(注) 10개로 구성. 핵심 수치:

| 항목 | 규격 |
| --- | --- |
| 외피 개구부 너비 | 8.34mm (오차 -0.02 / +0.06mm) |
| 외피 개구부 높이 | 2.56 ± 0.04mm |
| 리셉터클 텅(tongue) 너비 | 6.690mm (오차 -0.055 / +0.045mm) |
| 리셉터클 텅(tongue) 길이 | 4.45 ± 0.10mm |
| 각 신호 접점 너비 | 0.25 ± 0.04mm |
| 신호 접점 길이(전도성 외피, 그림3) | 데이터 등 3.50 ± 0.15mm; 전원·접지 접점은 데이터 접점보다 0.50 ± 0.20mm 길게 |
| 신호 접점 길이(비전도성/외피無, 그림4) | 데이터 등 3.45 ± 0.10mm; 전원 접점은 데이터보다 0.20 ± 0.10mm, 접지 접점은 전원보다 0.35 ± 0.10mm 길게 |

접촉 치수 적용 규칙(주 8~10): 외피가 전도성이면 그림3 또는 그림4 치수 사용 가능; 외피가 비전도성이면 그림4 치수만 사용(그림3 불가); 외피가 전혀 없는 경우 그림3 또는 그림4 사용하되, 전도성 외피를 효과적으로 제공하지 않는 구현에서는 그림4 치수 리셉터클 사용.

- [ ] 유선 충전 포트가 USB C형 리셉터클 커넥터인가(제3조).
- [ ] [별표 1] 치수 규격(개구부 너비/높이, 텅 너비/길이, 신호 접점 너비/길이) 적합 여부 확인.
- [ ] 외피 전도성/비전도성/무외피 구분에 따른 접촉 치수(그림3/그림4) 적용 규칙 준수.
- [ ] 시행일(2026-11-06) 이후 제조·수입·판매 제품 대응 계획 수립.
