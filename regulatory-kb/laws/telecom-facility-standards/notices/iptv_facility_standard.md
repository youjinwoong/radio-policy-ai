# 요약

이 고시는 「인터넷 멀티미디어 방송사업법」 제14조의2에 근거하여 **IPTV(인터넷 멀티미디어 방송) 제공사업자 설비, 가입자 단말장치의 기술적 조건, 서비스 품질**에 관한 기술기준을 제시하는 국립전파연구원 고시(제2022-19호, 2022-10-17 일부개정)이다. 전 16개 조문으로 구성되며, 핵심설비 관리(예비설비·월 점검), 전원·보호기·접지, 망 접속 규격, 서비스/시스템 정보, 음성·영상 코덱 및 다중화 규격, 제한수신(CAS), 콘텐츠 보안(불법복제방지), 네트워크 품질기준을 규정한다.

전원·보호기·접지 등 물리적 설비 요건은 상위 「방송통신설비의 기술기준에 관한 규정」(제7조·제10조)을 준용하고, 서비스·코덱·CAS 등 IPTV 특유의 기술 규격은 국제표준(MPEG, ITU-T, ISO/IEC, IETF)과 국내 방송통신단체표준(TTA)을 인용한다.

실무 영향: IPTV 사업자는 (1) 핵심설비 예비설비 구비 및 **매월 적합성 점검·기록·관리**, (2) 정전 시 **최대부하전류 기준 3시간 이상** 예비전원 확보, (3) 지정 코덱(AAC/AC-3, H.264/H.265, MPEG-2 TS)·CAS·불법복제방지 기능 지원, (4) **패킷 전달지연 100ms·손실률 10⁻³·지연편차 50ms 이하**의 네트워크 품질 준수가 요구된다.

# 적용 범위

- **대상**: 「인터넷 멀티미디어 방송사업법」 제4조에 따라 허가를 받아 광대역종합정보통신망 등을 이용, **IP 방식으로 실시간 IPTV를 제공하는 사업자**.
- **적용 사항**: ① IPTV 제공사업자 설비, ② 가입자 단말장치의 기술적 조건, ③ 서비스 품질.
- **용어**: 제3조 제1항의 17개 정의(핵심설비, 가입자 단말장치, 멀티캐스트, 제한수신, IP, 패킷 및 패킷 지연·손실·편차 등) 적용, 그 밖은 법·방송법·방송통신발전기본법·전기통신사업법·설비 기술기준 규정에 따름.

# 주요 기술기준(구조화)

| 조 | 제목 | 요지 (실무 포인트) |
|----|------|---------------------|
| 제1조 | 목적 | 인터넷 멀티미디어 방송사업법 제14조의2에 따른 IPTV 방송통신설비 기술기준 제시 |
| 제2조 | 적용범위 | 허가 IPTV 사업자 설비·단말 기술조건·서비스 품질에 적용 |
| 제3조 | 정의 | 핵심설비, 가입자 단말장치, 멀티캐스트, 제한수신, IP/패킷 등 17개 용어 정의 |
| 제4조 | 핵심설비의 관리 | ① 고장 시 교체·절체용 **예비설비를 방송국사 내 구비**, 운용상태 감시설비 구축·운용 ② **매월 적합성 점검·기록·관리** |
| 제5조 | 전원설비 | 상용전원설비 + **최대부하전류 기준 3시간 이상** 공급 예비전원설비 설치(설비 기술기준 규정 **제10조** 준용) |
| 제6조 | 보호기 및 접지 | 과전압·과전류·벼락 등 대비 보호기·접지설비 설치(설비 기술기준 규정 **제7조** 준용) |
| 제7조 | 망 접속 규격 | 단말 물리 망접속 **IEEE 802.3i/802.3u/802.3ab(10/100/1000 Mbps)**; 무선접속 추가 시 신고면제 무선국용 무선설비 기술기준 제7조5항·7항 준용 |
| 제8조 | 서비스 및 시스템 정보 | ① 8개 정보 항목을 **동일 형식으로 제공** ② TTA **TTAK.KO-08.0027/R3**, **TTAK.KO-08.0028/R2** 또는 디지털 유선방송 정합표준 준수 ③ 표준 개정 시 개정본 적용 가능 |
| 제9조 | 멀티캐스트 채널 전환 규격 | IETF **RFC 2236(IGMP)** 또는 **RFC 3810(MLD)** 준수 |
| 제10조 | 음성신호 압축·복호화 규격 | **ISO/IEC 13818-7 AAC 및 AC-3** 형식 준수 |
| 제11조 | 영상신호 압축·복호화 규격 | **ITU-T H.264(AVC, ISO/IEC 14496-10)** 또는 **ITU-T H.265(HEVC, ISO/IEC 23008-2)** 형식 준수 |
| 제12조 | 전송스트림 다중화·역다중화 규격 | **ISO/IEC 13818-1 MPEG-2 전송스트림(TS)** 형식 준수 |
| 제13조 | 제한수신 | ① 설비·단말은 CAS 기능 지원 ② 인증·제한수신키 안전 수신·정상 시청 3기능 제공(SW/HW) ③ 제2항 충족 또는 TTA **TTAK.KO-08.0023(iCAS)**·디지털 유선방송 정합표준 충족 ④ (삭제) |
| 제14조 | 콘텐츠 보안 | 보안 요구 콘텐츠에 대한 **불법복제방지** 기능 지원 |
| 제15조 | 네트워크 품질 | 패킷 전달지연 **100ms 이하**, 손실률 **10⁻³ 이하**, 지연편차 **50ms 이하** |
| 제16조 | 재검토기한 | 2020-01-01 기준 매 3년(3년째 12/31까지) 타당성 검토 |
| 부칙 | 제2022-19호 | 고시한 날(2022-10-17)부터 시행 |

## 서비스 및 시스템 정보 8개 항목 (제8조제1항)
1. 방송서비스 이름  2. 방송채널 번호  3. 서비스 ID  4. 멀티캐스트 주소 및 포트 번호  5. 프로그램 제목  6. 프로그램 시작시간  7. 프로그램 상영시간  8. 프로그램 시청등급

## 네트워크 품질기준 (제15조)
| 항목 | 기준 |
|------|------|
| 패킷 전달 지연 (IP packet transfer delay) | 100 ms 이하 |
| 패킷 손실률 (IP packet loss ratio) | 10⁻³ 이하 |
| 패킷 지연 편차 (IP packet delay variation) | 50 ms 이하 |

# 실무 체크리스트

- [ ] 핵심설비 고장 대비 **예비설비를 방송국사 내 구비**, 운용상태 감시설비 구축.
- [ ] 핵심설비 기술기준 적합성 **매월 점검·기록·관리** (제4조).
- [ ] 예비전원 **최대부하전류 기준 3시간 이상** 공급 능력 확보 (제5조, 규정 제10조).
- [ ] 보호기·접지설비 설치 (제6조, 규정 제7조).
- [ ] 단말 망접속 **IEEE 802.3i/u/ab(10/100/1000M)** 준수, 무선 추가 시 해당 무선설비 기술기준 준수.
- [ ] 서비스/시스템 정보 8개 항목 동일 형식 제공, TTA 표준(08.0027/R3, 08.0028/R2) 준수.
- [ ] 채널 전환 IGMP(RFC 2236)/MLD(RFC 3810) 지원.
- [ ] 코덱 준수: 음성 **AAC + AC-3**, 영상 **H.264 또는 H.265**, 다중화 **MPEG-2 TS**.
- [ ] CAS 기능(인증·키 안전수신·정상시청) 지원 및 iCAS(TTAK.KO-08.0023) 등 충족.
- [ ] 보안 콘텐츠 **불법복제방지** 기능 지원.
- [ ] 네트워크 품질 **지연 100ms / 손실 10⁻³ / 편차 50ms 이하** 상시 유지·측정.
