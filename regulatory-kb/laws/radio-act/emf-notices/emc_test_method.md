# 요약

적합성평가 대상기자재의 전자파적합성(EMC) 시험을 실제로 수행할 때 적용해야 하는 **시험표준(KS·KN·IEC 등)을 항목별로 지정**한 국립전파연구원 공고이다. 「방송통신기자재등의 적합성평가에 관한 고시」 제4조제3항이 근거이며, 2025년 7월 11일 일부개정되어 발령 후 1개월이 지난 2025년 8월 12일부터 시행된다. 구조는 (1) 총칙, (2) 세부 시험방법 — ① 측정기구·시험장·측정방법 등 일반사항(제3조, 22개 KS 표준), ② 대상기기별 세부 시험방법(제4조, 28개 항목 + 무선설비 15개 세분류) — 로 구성된다. 즉 「전자파적합성 기준」(제2025-22호)이 각 기기가 만족해야 할 허용기준(한도값)을 정한다면, 이 공고는 그 기준 적합 여부를 측정하는 방법과 인용표준을 정한다. 다만 별표(태양광 전력변환기, 전기철도기기류 시험방법)는 본문에 첨부파일로 링크되어 있어 원문 PDF/첨부에서 확인해야 한다.

# 적용 범위

- 「방송통신기자재등의 적합성평가에 관한 고시」 제3조에 따른 **적합성평가 대상기자재의 전자파적합성 시험**(적합성 시험)에 적용(제2조).
- EMC 시험을 수행하는 지정시험기관·제조사·수입사의 시험 절차 기준.

# 주요 내용(구조화)

## 제1장 총칙(제1조·제2조)
- 목적: 적합성평가 대상기자재의 EMC 시험방법에 관한 사항 규정.
- 적용범위: 위 고시 제3조 대상기자재의 적합성 시험.

## 제2장 세부 시험방법

### 제3조 일반사항 세부 시험방법(측정기구·시험장·공통 측정방법)
- 측정기구·보조장비: KS C 9816-1-1~1-5(측정기구, 전도성/방해전력/방사성 방해 측정용 보조장비, 30 MHz~1000 MHz 안테나 교정시험장).
- 방해 측정방법: KS C 9816-2-1~2-5(전도성 방해, 방해전력, 방사성 방해, 내성, 대형기기 현장 측정).
- 전원계통 방출/내성: KS C 9610 계열(고조파 전류, 전압변동·플리커, 정전기방전(ESD, 4-2), 방사성 RF 내성(4-3), EFT/버스트(4-4), 서지(4-5), 전도성 RF 내성(4-6), 전원주파수 자기장(4-8), 펄스 자기장(4-9), 순간전압강하·정전(4-11), 저주파 내성(2-2·2-4) 등).

### 제4조 대상기기별 세부 시험방법(총 28개 항 + 무선설비 15종)
- 대표 예시: ISM기기 장해방지(KS C 9811), 의료기기 내성(KS C IEC 60601-1-2), 아크용접기(KS C 9974-10), 태양광 전력변환기(별표1 KN 62920), 자동차·내연기관(KS C 9990/9995), 가정용 전기·전동기기(KS C 9814-1/-2), 조명기기(KS C 9815/9547), 전기철도기기(별표2 KN 50/KN 51), 전력선통신(KS X 3141).
- **무선설비기기류(제11항)**: 공통(KS X 3124), 무선데이터통신 소출력(3126), 이동통신 단말·보조기기(3129), 이동통신 기지국·중계기·보조기기(3135), 위성휴대통신(3139) 등 15종 세분류. 다만 해상업무용·항공기 탑재 무선설비는 제외.
- 기타: 무정전전원장치, 저압개폐·제어장치, 멀티미디어기기(장해방지 9832/내성 9835), 가변속 전력구동기기, 승강기, 해상·선박용, 무선전력전송기기(9143), 환경별 일반 장해방지·내성(KS C 9610-6 계열), 계량기·소방용품·항공기 탑재기기·전기자전거 등.

## 부칙(제2025-50호, 2025.7.11.)
- **시행일**: 발령 후 1개월 경과한 날(2025.8.12.).
- **경과조치**: 시행 당시 종전 시험방법 제3조·제4조에 따라 적합성평가 시험을 한 경우 종전 규정에 따름.

# 실무 체크리스트

- [ ] 시험 대상기기의 종류를 확인하고 제4조에서 해당 항의 대상기기별 시험표준을 특정.
- [ ] 일반 측정기구·시험장은 제3조의 KS 계열 표준으로 셋업.
- [ ] 무선설비는 제4조제11항의 15종 세분류 중 정확한 표준(KS X 31xx) 매칭 및 해상·항공 제외 여부 확인.
- [ ] 별표1(태양광), 별표2(전기철도) 대상이면 원문 첨부파일의 상세 방법 확보.
- [ ] 2025.8.12. 이전 시험 완료 건은 종전 규정 적용 가능(경과조치).
