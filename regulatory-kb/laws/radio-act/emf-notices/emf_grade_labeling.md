# 요약

「전파법」 제47조의2제1항에 따라 전자파 노출 정도를 일반인이 알아볼 수 있도록 등급화·표시하도록 한 고시. 두 체계로 구성된다. (1) **전자파강도 등급**: 「전파법 시행령」 제65조에 따른 전자파강도 보고대상 무선국을 대상으로, 측정값을 인체보호기준 대비 비율로 4개 등급(1등급·2등급·주의등급·경고등급)으로 구분하고, 무선국 시설자가 펜스·안테나설치대·무선설비 등에 등급 라벨을 표시하도록 의무화([별표1·2]). (2) **전자파흡수율(SAR) 등급**: 이동통신 휴대용 무선설비, 무선랜(WAS)·무선데이터통신용 특정소출력 무선기기를 대상으로 SAR 측정값을 1등급(≤0.8)·2등급(0.8~1.6, W/kg 1g 평균)으로 구분하고, 제조·수입자가 제품 본체·포장·설명서·홈페이지 등에 등급 또는 측정값을 표시하도록 의무화([별표3·4]). 등급 판정에 쓰이는 기준값은 상위 「전자파인체보호기준」 고시를 따른다. 시행일 2018.4.19.(공포 후 6개월).

# 적용 범위

- **전자파강도 등급 표시대상**: 「전파법 시행령」 제65조에 따른 전자파강도 보고대상 무선국(제2조제1항). 표시의무자는 무선국 시설자.
- **SAR 등급/측정값 표시대상**(제3조제1항): 「전기통신사업용 무선설비의 기술기준」 제4조 이동통신용 무선설비 중 휴대용 무선설비, 「신고하지 아니하고 개설할 수 있는 무선국용 무선설비의 기술기준」 제7조제5항 무선랜 포함 무선접속시스템용(WAS) 특정소출력 무선기기, 같은 조 제7항 무선데이터통신시스템용 특정소출력 무선기기. 표시의무자는 제조자·수입자("제조자 등").
- 다중 모드 무선기기: 측정된 SAR값 중 최대값에 해당하는 등급/측정값으로 표시(제3조제3항 단서).

# 주요 내용(구조화)

## 전자파강도 등급기준 [별표1] (제2조)

| 등급명 | 기준 |
|---|---|
| 1등급 | 측정값 ≤ 일반인 전자파강도기준의 50% |
| 2등급 | 일반인 기준의 50% < 측정값 ≤ 일반인 기준 |
| 주의등급 | 일반인 기준 < 측정값 ≤ 직업인 기준 |
| 경고등급 | 직업인 기준 < 측정값 |

- 비고: 일반인·직업인 전자파강도기준은 「전자파인체보호기준」 고시를 따름.

## 전자파강도 등급 표시(제2조제3항, [별표2])
- 표시위치(택1): ① 펜스·울타리·철조망·안테나설치대 ② 해당 무선설비 ③ 기타 일반인이 쉽게 식별 가능한 위치.
- 라벨 구성: 해당 설비의 등급·측정값·측정일 정보 포함.
- 라벨 색상(4원색 CMYK): 1등급 연두(C50/M0/Y100/K0), 2등급 파랑(C70/M15/Y0/K0), 주의등급 주황(C0/M35/Y100/K0), 경고등급 빨강(C10/M98/Y92/K5).
- 라벨 크기: 기본 10cm(가로)×10cm(세로), 동일비율로 축소·확대 가능.
- 견고하게 인쇄·각인 등으로 부착.

## 전자파흡수율(SAR) 등급기준 [별표3] (제3조, 단위 W/kg, 1g 평균)

| 등급명 | 기준 |
|---|---|
| 1등급 | SAR 측정값 ≤ 0.8 |
| 2등급 | 0.8 < SAR 측정값 ≤ 1.6 |

- 비고: SAR 측정값은 「전자파흡수율 측정기준」 고시에 따라 머리에 근접하여 시험된 값.

## SAR 등급 표시(제3조제3항, [별표4])
- 표시위치(택1): ① 제품 본체(탈착 필름용지 부착 포함) ② 제품 포장상자 ③ 사용자 설명서 표지 ④ 휴대전화 내 정보 메뉴 또는 별도 안내문 ⑤ 제조사 홈페이지 등(단 ①~④에 확인 가능한 인터넷 주소 표시 필요).
- 등급 표시 시 SAR 측정값을 함께 표시 가능(제3조제4항).
- 라벨: 전자파흡수율 등급(1·2등급) 및 안내문구 필수 표시, 도안 비율은 제조자 자율, 색채 흑백 또는 컬러.
- 측정값 표시 시: 노출한계치(1.6 W/kg) 및 최대 SAR 측정값 필수 표시. 우리나라는 국제권고기준(2 W/kg)보다 엄격한 1.6 W/kg 적용 안내 포함.

## 사후관리(제4조)
- 장관은 전자파강도·SAR 등급 표시 여부 및 허위 표시 등에 대한 조사 실시 가능.

## 재검토기한(제5조)
- 2015년 1월 1일 기준 3년마다(매 3년째 12월 31일까지) 타당성 검토.

## 부칙
- 시행일: 공포 후 6개월 경과한 날(2018.4.19.).
- 경과조치: 시행 전 적합성평가를 받았거나 신청한 기자재는 종전 규정에 따름.

# 실무 체크리스트

- [ ] 전자파강도 보고대상 무선국(시행령 제65조) 여부 확인 → 해당 시 등급 산정·라벨 표시 의무.
- [ ] 무선국 등급은 인체보호기준 대비 측정값 비율로 산정(50% 이내=1등급 등), 라벨에 등급·측정값·측정일 기재.
- [ ] 라벨을 접근 가능 위치(펜스·안테나설치대 등)에 견고하게 부착, 규정 색상·크기 준수.
- [ ] 휴대용 무선기기 제조·수입 시 SAR 등급(1등급 ≤0.8, 2등급 ≤1.6) 산정 및 표시.
- [ ] SAR 다중 모드 기기는 최대 측정값 기준으로 등급 표시.
- [ ] SAR 측정값 표시 시 노출한계치 1.6 W/kg 병기, 제조사 홈페이지 등에서 상세 확인 경로 제공.
- [ ] 허위·미표시 시 사후관리 조사 대상임을 인지.
