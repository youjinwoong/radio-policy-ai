# 요약

「전파법」 제47조의2제1항에 따라, 상위 「전자파인체보호기준」이 정한 전자파강도기준의 적합성을 확인하기 위한 측정방법·절차의 표준을 정한 고시. 측정기기의 일반적 조건(동작범위·차폐·연속동작 8시간·실효치와 첨두값 측정 등)과 측정프로브 크기 요건, 교정 불확정도(±2dB 이내, 최대 ±4dB), 측정조건(접근 가능한 모든 장소·최악 노출조건 선택), 저주파(100kHz 미만)·고주파(100kHz~300GHz) 전자기장 측정방법을 규정한다. 특징적으로 이 고시는 세부 측정방법을 국가표준(KS)에 위임한다 — 무선국은 KS C 3367, 항행/탐지 육상국은 KS X 3278, 가전기기는 KS C 3369, 6GHz 이상 이동통신 무선설비 전력밀도는 KS C 3390, 전기차 무선전력전송은 KS C 3380을 각각 적용한다. 종전 별표·별지 서식은 모두 삭제되었다(2023 개정).

# 적용 범위

- 과기정통부장관이 고시한 「전자파인체보호기준」의 **전자파강도기준 적합성 평가**에 적용(제3조제1항).
- 세부 측정은 국가표준(KS)을 적용:
  - 무선국의 전자파강도 측정: KS C 3367(무선국의 전자파 인체노출 적합성 평가방법). 단 무선항행육상국·무선탐지육상국(실험국 포함)은 KS X 3278(RF 펄스형 신호 무선국 측정방법).
  - 가전기기: KS C 3369(가전기기 및 유사기기 자기장 측정방법).
  - 6GHz 이상 이동통신용 무선설비 전력밀도: KS C 3390(6GHz 이상 휴대용 송신 무선설비 전력밀도 측정방법).
  - 전기자동차 무선전력전송기기: KS C 3380(저주파수 자기장 인체노출량 측정방법).

# 주요 내용(구조화)

## 정의(제2조)
- 측정기준, 전기장강도·자기장강도·자속밀도·전력밀도, 저주파수 대역(100kHz 미만)·고주파수 대역(100kHz~300GHz), 원거리장/근거리장 영역, 자유공간 조건, 편파(선형/타원/원), 다중 복사원, 프로브, 실횻값(rms)·첨둣값, 합성 전자기장, 측정 불확정도, 실효두께(TE = T×εr^1/2) 등.

## 측정기기의 일반적 조건(제4조)
- 충분한 동작범위·주파수대역, 측정기기·전원선·케이블 적절히 차폐.
- 저주파수대역 측정기기: 내장전원으로 8시간 이상 연속동작 가능.
- 전기장·자기장 성분의 실횻값·첨둣값 측정 가능.
- 프로브 크기: 저주파수대역 단축프로브 단면적 <0.01㎡, 3축 프로브 최대 <0.2m / 고주파수대역 프로브 크기는 파장의 1/4보다 작거나 <0.1m(1MHz 이하 고주파는 자유공간조건 최대 ≤0.2m).

## 측정조건(제5조)
- 접근 가능한 모든 장소에서 측정, 여러 노출조건이면 최악 조건 선택.
- 프로브-발생원 충분히 이격, 측정 중 휴대기기 전원 차단, 프로브 주변 산란체 없어야 함(불가피 시 결과서에 기록).

## 교정 및 불확정도(제6조)
- 교정 유효기간 이내 기기 사용, 수리 후 즉시 교정.
- 교정 불확정도 ±2dB 이내, 초과 시 보고서 명시, 최대 ±4dB 초과 불가.

## 저주파 전자기장 측정방법(제8조)
- 3축 등방성프로브로 합성전자기장 최댓값 측정(선형/타원편파 파악 시 단축프로브 가능).
- 전기장강도 측정 시 프로브-주변 산란체 거리 ≥ 프로브 크기의 5배(자기장강도는 거리 제한 없음).
- 전력선 아래 측정: 지표면 위 1m 높이, 프로브는 전기장 수직성분 측정 위치, 이동가능 물체와 물체높이 3배 이상·지상 고정물체와 1m 이상 이격.

## 고주파 전자기장 측정방법(제9조)
- 3축 등방성프로브로 합성전자기장 측정.
- 근거리장: 전기장·자기장강도 모두 측정 / 원거리장: 하나 측정 후 나머지 계산 가능.
- 금속부는 흡수체로 둘러싸고, 유전체 지지대는 낮은 손실탄젠트(tanδ≤0.05)·낮은 상대유전율(εr≤5.0) 또는 실효두께 파장의 1/4 이하.
- 프로브-복사원·산란체 거리 ≥20cm.

## 보칙·재검토(제10~13조)
- 측정결과서는 해당 국가표준 서식에 따라 작성(제10조).
- 자속밀도·전력밀도는 측정 자기장/전자기장 강도로부터 계산 가능(제11조, 근거리는 KS C 3390 참조).
- 신규 통신서비스 등 전파환경 변화 시 원장이 세부지침 별도 제정 가능(제11조제2항).
- 재검토기한(제12조): 2024.1.1. 기준 3년마다. 규제 재검토(제13조): 2022.1.1. 기준 매 3년.

# 실무 체크리스트

- [ ] 측정 대상(무선국/가전기기/6GHz 이상 이동통신/전기차 무선전력전송)에 맞는 KS 국가표준 식별·적용.
- [ ] 측정기기 교정 유효기간·불확정도(±2dB 이내, 최대 ±4dB) 확인, 초과 시 보고서 명시.
- [ ] 접근 가능한 모든 장소 측정, 여러 노출조건 시 최악 조건 선택.
- [ ] 저주파/고주파에 따른 프로브 크기·이격거리(전기장 5배, 고주파 20cm 등) 준수.
- [ ] 전력선 아래 측정 시 지표면 1m 높이·수직성분·이격거리 규정 준수.
- [ ] 측정결과서를 해당 KS 표준 서식으로 작성.
