# 요약

무선국의 무선설비를 변경할 때 원칙적으로 변경허가가 필요하나, 이 고시는 전파법 시행령 제31조제4항제10호(무선국) 및 제76조제4항제3호(전파응용설비)의 위임에 따라 변경허가를 받지 않아도 되는 대상을 열거한다. 대상은 간이무선국 무선설비기기, 라디오부이·라디오존데, 주파수측정장치, 무선방위측정기 등 특정 기기와, 긴급수리를 위한 의무항공기국 무선설비의 동일 형식·성능 교체, 다중 차폐시설 내 전파응용설비의 동일 성능 교체·증설·이설이다. 다만 후자 두 경우(제2조제6호·제7호)는 변경허가는 면제되나 교체·증설·이설 완료 후 도래하는 정기검사 전까지 관할 허가관청에 "허가변경명세서"를 제출해야 한다.

# 적용 범위

- 근거: 전파법 시행령 제31조제4항제10호(무선기기), 제76조제4항제3호(전파응용설비).
- 대상: 아래 7개 유형의 무선기기 및 전파응용설비.
- 소관: 중앙전파관리소(전파관리과).
- 성격: 변경허가 면제 대상 지정(면제이지 신고·검사 면제는 아님 — 제6호·제7호는 사후 명세서 제출 의무 존재).

# 주요 내용(구조화)

## 변경허가가 필요하지 아니한 대상 (제2조)

| 호 | 대상 | 비고 |
|---|---|---|
| 1 | 간이무선국의 무선설비기기 | |
| 2 | 라디오부이 | |
| 3 | 라디오존데 | |
| 4 | 주파수측정장치 | |
| 5 | 무선방위측정기 | |
| 6 | 긴급수리를 위해 이미 허가받아 설치된 설비와 동일 형식·동일 성능의 설비로 교체하는 경우로서, 전파법 제22조제2항에 따른 의무항공기국의 무선설비 | 사후 허가변경명세서 제출 대상 |
| 7 | 지속적 가동을 위해 이미 허가받은 설비와 동일 성능의 설비를 교체·증설·이설하는 경우로서, 전자파차단이 양호한 다중 차폐시설(전자파차폐 시설을 갖춘 건물 내에 전자파차폐 설비를 추가 설치한 경우)을 갖춘 건물 내에 설치된 전파응용설비 | 사후 허가변경명세서 제출 대상 |

## 허가변경명세서 제출 (제3조)

- 대상: 제2조제6호·제7호의 무선기기 및 전파응용설비.
- 시점: 교체·증설·이설 완료 이후 도래하는 정기검사 전까지.
- 제출처: 관할 허가관청.
- 근거: 전파법 제58조제3항, 같은 법 시행령 제33조제4항, 같은 법 시행규칙 제8조제4항.

## 재검토기한 (제4조)

- 2021.7.1을 기준으로 매 3년이 되는 시점(매 3년째 6.30까지)마다 타당성 검토.

# 실무 체크리스트

- 무선설비 변경 예정 시 해당 설비가 제2조 각 호(간이무선국 기기, 라디오부이·존데, 주파수측정장치, 무선방위측정기 등)에 해당하는지 우선 확인 → 해당 시 변경허가 불요.
- 의무항공기국 무선설비 긴급 교체(제6호): "동일 형식·동일 성능" 요건 충족 여부 확인, 교체 후 정기검사 전까지 허가변경명세서 제출.
- 전파응용설비 교체·증설·이설(제7호): "다중 차폐시설(이중 전자파차폐)" 건물 요건 및 "동일 성능" 요건 확인, 정기검사 전까지 허가변경명세서 제출.
- 제6호·제7호는 변경허가만 면제될 뿐 명세서 제출 의무가 있으므로 정기검사 일정과 연계하여 관리.
