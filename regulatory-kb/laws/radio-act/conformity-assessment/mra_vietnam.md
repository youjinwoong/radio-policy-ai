# 요약

대한민국 방송통신위원회(당시 소관, 현재 과학기술정보통신부·국립전파연구원으로 기능 승계)와 베트남 정보통신부(MIC) 간에 체결된 방송통신기자재등 상호인정협정(MRA)을 고시로 공표한 것이다. 1998년 APEC 정보통신장관회의에서 승인된 「APEC 정보통신기기의 적합성 평가에 대한 상호인정협정」에 근거하며, 방송통신위원회고시 제2011-41호로 제정되어 2011.8.2. 시행되었다.

본 협정은 1단계 MRA로, 양국이 상대국 기술규정에 따라 기술기준 적합성을 시험할 시험기관을 상호 지정·통보·승인하는 구조이다. 대한민국 방송통신위원회 전파연구소(현 국립전파연구원)는 베트남 기술규정에 따라 시험기관을 지정·통보하고 베트남 MIC 과학기술국이 승인하며, 역방향도 동일하다. 이로써 상대국 인정 시험기관이 발행한 시험성적서를 자국 적합성평가에 활용할 수 있다. (한-캐나다 MRA가 인증기관까지 상호 인정하는 2단계인 것과 달리, 본 협정은 시험기관 상호 인정 단계이다.)

# 적용 범위

- 대상기자재: ①대한민국 - 전파법 제58조의2제1항에 따른 방송통신기자재등, ②베트남 - 별표 1에 따른 기자재(단말기기·무선기기·IT기기).
- 시험기관 지정절차·방법: 「APEC 정보통신기기의 적합성 평가에 대한 상호인정협정」을 따름.
- 적용 기술기준: 베트남 측은 별표 1, 대한민국 측은 별표 2에 정한 기술기준을 적용.

# 주요 내용(구조화)

## 1. 적합성 평가 시험기관 상호 인정 (1단계)
- 전파연구소(현 RRA)가 베트남 기술규정에 따라 시험기관을 지정 → 베트남 MIC에 통보 → MIC 과학기술국 승인.
- MIC 과학기술국이 한국(방통위·전파연구소) 기술규정에 따라 시험기관을 지정 → 대한민국 전파연구소에 통보 → 전파연구소 승인.

## 2. 적용범위
- 대상기자재(양국), 시험기관 지정절차(APEC MRA 준용), 기술기준(별표 1·2).

## 부칙 (제2011-41호, 2011.8.2.)
- 시행일: 고시한 날(2011.8.2.).
- 기술기준의 효력: 시행 후 방통위·전파연구소 또는 베트남 MIC 기술기준이 변경되면 변경된 기술기준 적용.
- 시험성적서 언어: 상호인정 받고자 하는 국가의 언어 원칙, 공용어로 영어 사용 가능. 다만 공용어(영어) 사용 성적서 제출 시 적합성평가 처리기간은 법정처리기간이 아닌 실제 소요기간으로 함.

# 별표·서식 목록

- [별표 1] 베트남 정보통신부 대상기자재와 기술기준:
  - 1. 단말기기 - PSTN 아날로그 가입자 인터페이스 단말기(TCN 68-188/190/193/196), 코드리스 전화 단말기(TCN 68-188/143/190/192/196), 기본요율접속(BRA) ISDN 단말기(TCN 68-189/190/193/196), GSM 이동국 Phase 2/2+(TCN 68-221), CDMA 이동국(TCN 68-222/245), PHS 단말기(TCN 68-223).
  - 2. 무선기기 - 2.4GHz 확산대역변조 무선기기(TCN 68-242), 9kHz~25MHz 근거리 무선기기(TCN 68-243), 아날로그TV 방송서비스용 송신기(TCN 68-246).
  - 3. IT기기 - 데스크탑 PC·서버, 노트북·포터블 컴퓨터, PDA, 라우터·허브·스위치·게이트웨이·브릿지·방화벽(모두 TCVN 7189).
  - ※베트남 MIC 기술기준은 www.mic.gov.vn 확인.
- [별표 2] 대한민국 방송통신위원회 및 전파연구소의 기술기준(18개): 적합성평가 고시, 시험기관 지정관리 고시, 방송통신설비 기술기준 규정(대통령령)·표준시험방법, 유선방송국 설비 기술기준, 단말장치 기술기준, 방송공동수신설비 설치기준, 무선설비규칙, 전자파 인체보호·흡수율·강도 측정기준 및 측정방법, 무선/유선설비 적합성평가 처리방법, 전자파장해방지·전자파보호 기준 및 시험방법 등. ※www.rra.go.kr 확인.

# 실무 체크리스트

- [ ] 베트남향 수출/베트남산 수입 기자재는 상대국 인정 시험기관 발행 시험성적서를 국내 적합성평가에 활용 가능한지 확인한다.
- [ ] 적합인증/적합등록 시 시험성적서 출처가 "국가 간 상호인정협정 체결국의 원장 인정 시험기관"에 해당하는지 확인(적합성평가 고시 제5조1항3호다목·제10조1항3호다목).
- [ ] 적합등록 신청서(별지 제5호) 등록방식에서 "FTA/MRA에 따른 적합등록" 해당 여부를 확인한다.
- [ ] 베트남 대상기자재는 별표 1의 TCN/TCVN 기술기준을, 국내는 별표 2 기술기준을 적용(변경 시 변경본).
- [ ] 영어(공용어) 성적서 제출 시 처리기간이 법정기간이 아닌 실제 소요기간임에 유의(부칙 제3조).
- [ ] 소관·집행 기능은 방송통신위원회 → 과학기술정보통신부·국립전파연구원으로 승계되었음을 인지(문서 문의처는 과기정통부 전파기반과).
