# 요약

전력선통신(PLC)설비가 다른 무선통신에 유해한 방해를 주지 않도록, 특정 주파수대역에서 일정 전계강도를 초과하는 운용을 금지하는 고시. 대상은 AM방송, 아마추어무선, 조난·긴급·안전통신(항공·해상), 해상업무용통신 대역이다. "전력선통신설비"란 전선로에 주파수 9kHz 이상의 전류가 흐르는 통신설비를 말한다. 금지는 절대적이 아니라 "설비로부터 3m 거리에서 측정한 전계강도가 기준값(6.3 또는 16 μV/m)을 초과하는 경우"에 한하는 조건부 규제이므로, 실무상 방사 전계강도 관리가 핵심이다.

# 적용 범위

- 대상 설비: 전선로에 9kHz 이상 전류가 흐르는 전력선통신(PLC)설비.
- 금지 성격: 아래 표의 대역에서 3m 거리 전계강도가 기준값을 초과하는 운용을 금지(기준 이하이면 운용 가능).
- 소관: 과학기술정보통신부(전파기반과).
- 근거 성격: 전파법령상 PLC설비의 타 통신 보호를 위한 운용 제한.

# 주요 내용(구조화)

## 운용을 금지하는 주파수대역 및 전계강도 기준 (제1조)

| 구분 | 운용 금지 주파수대역 | 초과 시 금지되는 전계강도 (3m 거리 측정) |
|---|---|---|
| AM방송 | 526.5 ~ 1,605.5 kHz | 6.3 μV/m 초과 |
| 아마추어무선 | 1,800~2,000 / 3,500~4,000 / 7,000~7,300 / 10,100~10,150 / 14,000~14,350 / 18,068~18,168 / 21,000~21,450 / 24,890~24,990 / 28,000~29,700 kHz | 16 μV/m 초과 |
| 조난통신 등 - 항공통신 | 2,850~3,025 / 3,400~3,500 / 6,525~6,685 / 8,815~8,965 / 10,005~10,100 / 13,260~13,360 / 17,900~17,970 kHz | 16 μV/m 초과 |
| 조난통신 등 - 해상조난통신 | 2,173.5~2,190.5 / 4,176.5~4,178.5 / 8,413.5~8,415.5 / 27,819.9~27,824.9 kHz | 옥외 운용 설비로부터 3m에서 16 μV/m 초과 |
| 조난통신 등 - 해상업무용통신(주) | 450 kHz ~ 30 MHz | 해안국(중계소 포함) 설치장소 1km 이내 지역에 한하여, 옥외 운용 설비로부터 3m에서 16 μV/m 초과 |

주(注): 해상업무용통신란의 "해안국"은 전파법 시행령 제26조·제27조에 따라 해상이동업무를 행하는 해안국(중계소 포함)이며, 구체적 목록은 [붙임]에 90개소가 열거됨.

## 정의 및 재검토

- 정의: "전력선통신설비" = 전선로에 주파수 9kHz 이상 전류가 흐르는 통신설비.
- 재검토기한(제2조): 2015.1.1 기준 3년마다(매 3년째 12.31까지) 타당성 검토.

# 별표·서식 목록

- [붙임] 조난·긴급·안전통신 및 인명안전 업무통신을 행하는 해안국 목록 (무선국 설치장소·위도·경도, 총 90개소). DSC·6MF8 및 항무(항무부산 등)·해경·수협·해운 계열 해안국 좌표 포함.

# 실무 체크리스트

- PLC설비 도입/운용 시 상기 5개 카테고리 대역에서 3m 거리 전계강도 측정치가 기준(AM 6.3 μV/m, 그 외 16 μV/m) 이하인지 확인.
- 해상업무용통신 대역(450kHz~30MHz)은 해안국 1km 이내 지역·옥외 설비에 한정 적용 → 설치 위치가 [붙임] 90개 해안국 반경 1km 내인지 좌표로 검증.
- 450kHz 미만 대역에서 전계강도 기준 초과 시에는 별도로 허가 대상 여부 확인 필요(관련 문서 참조).
- 재검토 주기(3년) 도래 시 개정 여부 모니터링.
