# 요약

주파수분배 변경으로 인해 더 이상 사용할 수 없게 되는 방송통신기자재(신고 없이 개설 가능한 비면허 무선국용 무선설비)의 이용자를 지원하고, 해당 기자재의 사용종료·제조수입판매종료 등을 이용자에게 표시·고지하는 방법을 정한 고시. 지원대상 요건과 신청·심사·통보 절차(각 30일 기준), 금전지원 시 소유권포기 절차, 스티커 표시 규격, 구두 고지 방법을 규정한다. 근거는 전파법 시행령 제10조의2 및 제77조의13.

# 적용 범위

- 대상 기자재: 전파법 제19조의2제2항에 따라 신고 없이 개설 가능한 무선국용 무선설비 중 주파수분배 변경으로 사용 불가하게 되는 방송통신기자재.
- 대상자: 지원 신청일 기준 기자재를 구매·사용 중이며 과기정통부장관이 공고한 자(대리인 포함).
- 규율 내용: (1) 이용자 지원 대상·신청·심사, (2) 주파수분배 변경 표시·고지 방법.
- 소관: 과학기술정보통신부(주파수정책과).
- 근거: 전파법 시행령 제10조의2, 제77조의13.

# 주요 내용(구조화)

## 용어 정의 (제2조)

| 용어 | 정의 |
|---|---|
| 기자재 | 전파법 제19조의2제2항에 따라 신고 없이 개설 가능한 무선국용 무선설비 중 주파수분배 변경으로 사용 불가하게 되는 방송통신기자재 |
| 이용종료일 | 주파수분배 변경이 적용되는 날의 전날 |
| 예고기간 | 주파수분배 변경 고시일부터 이용종료일까지 |

## 지원대상 및 제외 (제3조)
- 대상: 신청일 기준 기자재 구매·사용 중이며 장관이 공고한 자(신청 대리인 포함).
- 제외: ① 수입·판매종료일 이후 구입자, ② 정상 작동하지 않는 기자재 보유자, ③ 적합성평가 미필 기자재 사용자, ④ 적합성평가 면제(법 제58조의3제1항 1·2호) 기자재 사용자.

## 지원방법 (제4조)
- 금전지원: 예고기간·평균 사용기간·내구연한·판매금액 등을 종합 고려하여 금액 또는 산정기준 결정.
- 금전지원 시 별지 제3호 소유권포기서와 기자재 제출 필요.
- 금액·산정방법 및 변경·개조 범위·방법은 장관이 공고.

## 신청·심사 절차 (제5조)

| 단계 | 기한/내용 |
|---|---|
| 제출 서류 | 이용자 지원 신청서(본인: 별지1호, 대리인: 별지2호) + 신분증 사본(우편·모사전송·인터넷 제출 시) |
| 제출처 | 비면허무선기기지원센터 또는 장관 지정 접수처(방문/등기우편/모사전송/인터넷) |
| 대상 여부 통보 | 접수기간 종료일부터 30일 이내 |
| 통보 내용 | 대상 시 지원방법·일정 / 비대상 시 사유 |
| 금전지원 확정 후 | 통보받은 날부터 30일 이내 소유권포기서·기자재 제출 |
| 부적합 통보 시 재신청 | 통보일부터 30일 이내 입증서류 첨부하여 재신청 가능 |

처리 흐름: 신청서 작성 → 접수 → 서류심사 → 조사·평가 → 결정 → 통보(금전지원은 금액 결정 → 통보 및 금전지급).

## 표시·고지 방법 (제3장, 제7~9조)
- 표시방법(제7조): 영 제77조의13제1항 표시사항을 기자재 전면·포장·홍보물에 인쇄 또는 스티커 부착, 이용자안내문 별도 동봉. 스티커 표시·크기는 별표 기준.
- 표시시기(제8조): 적합성평가를 받은 날 기준. 단, 주파수분배 변경 고시일이 더 늦으면 그 고시일 기준.
- 고지방법(제9조): 구매·대여받으려는 자가 이해하도록 구두 설명. 전자상거래 등 구두 곤란 시 눈에 잘 보이도록 게시하는 방법으로 갈음 가능.

## 스티커 규격 (별표)

| 항목 | 기준 |
|---|---|
| 표시 항목 | 사용 종료일(YYYY.MM.DD.), 제조·수입·판매 종료일(YYYY.MM.DD.) + "사용 종료일 이후 사용 시 법에 위반됩니다" 문구 |
| 크기 | 80mm(가로) × 20mm(세로) 기본, 50%~150% 범위 내 확대·축소 가능 |
| 색상 | 흰색 또는 노란색 바탕에 검정 글씨 |

## 재검토기한 (제10조)
2015.1.1 기준 3년마다(매 3년째 12.31까지) 타당성 검토.

# 별표·서식 목록

- [별표] 스티커의 표시 및 크기 등(제7조 관련)
- [서식 1] 이용자 지원 신청서
- [서식 2] 이용자 지원 신청서(대리인 신청시)
- [서식 3] 소유권포기서(금전지원 신청시)

# 실무 체크리스트

- 주파수분배 변경 예정 대역에 자사 비면허 방송통신기자재가 해당하는지 확인 → 이용종료일·예고기간 산정.
- 지원 신청 시 지원대상 제외사유(수입판매종료일 후 구입, 적합성평가 미필/면제 등) 해당 여부 사전 점검.
- 금전지원 선택 시 소유권포기서(서식3)와 기자재 제출 준비, 통보 후 30일 기한 준수.
- 제조·수입·판매자는 스티커(80×20mm, 흰색/노란색 바탕 검정 글씨) 및 이용자안내문 동봉, 구두 고지(전자상거래는 게시) 이행.
- 표시시기: 적합성평가일과 주파수분배 변경 고시일 중 늦은 날 기준.
